Diabetic Retinopathy Detection

https://www.kaggle.com/datasets/mariaherrerot/aptos2019


This project was made on Google Colab to detect Diabetic Retinopathy (DR) on the APTOS 2019 dataset. Follow the steps below to set up and run the project.


- Setup Instructions

Google Drive Setup:

Upload the APTOS 2019 dataset ZIP file to your Google Drive.
Place it in: /content/drive.

Storage Requirements:

8 GB for images resized to 256x256.
8 GB for images resized to 380x380.


Dependencies:

Required Python libraries will be installed during execution.
